# Rhythm-Circuit
Repository for the C++ code of the Arduino board of a small rhythm game project.
The LCD uses two stripes to display a row of notes moving leftwards accross the screen. Buttons must be pressed when the notes reach the left edge of the screen. The left button corresponds with the top row, the right the bottom row.
A score is calculated based on a player's ability to press the button at the correct time.
The rows can be changed by selecting the mirrored option.
The speed of the track can be selected afterwards on the multiplier select screen.
As per current configuration, the string for the two note strips is randomised.
